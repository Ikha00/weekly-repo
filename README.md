# weekly-repo
Repo for 1 week.
Projek ini dibangunkan sebagai latihan selama seminggu untuk memahami asas
GitHub, struktur projek web dan penggunaan HTML & CSS.
## Penerangan Projek
Projek ini mengandungi beberapa halaman web ringkas dengan reka bentuk asas.
Fokus utama adalah:
- pengurusan repository
- penggunaan branch
- design menggunakan css
## Cara Buka Projek
1. Clone repository dari GitHub
2. Buka folder projek menggunakan Visual Studio Code
3. Klik dua kali pada `index.html` atau buka menggunakan browser.
